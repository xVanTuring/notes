# 总结

## 第二章
8086 内部结构：算数逻辑单元ALU，累加器，专用和通用寄存器，指令寄存器，指令译码器
以及定时和控制电阻等组成

通用寄存器
* AX： 累加器
* BX： 基址寄存器
* CX： 计数寄存器
* DX： 数据寄存器

地址指针寄存器
* SP: 栈指针寄存器
* BP：基址指针寄存器

变址寄存器
* SI： 源变址寄存器
* DI： 目的变址寄存器

段寄存器
* CS: IP
* SS: SP/BP
* DS: BX, SI, DI, 常数(8/16bit)
* ES: DI

指令指针寄存器 IP

标志寄存器 FLAG


物理地址 = 段基址 \* 10H + 偏移地址


时钟周期是微处理器动作最小的时间单位， 两个时钟脉冲上升沿之间的时间间隔
