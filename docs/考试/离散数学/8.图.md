## 图的连通性
### 通路
$$\Gamma = v_{i_0} e_{j_1} v_{i_1}e_{j_1} v_{i_2}... e_{j_n} v_{i_n}$$

称之为 连接 $v_{i_0}$ 到 $v_{i_n}$的通路

| 分类     | 边都不同 | 起点终点相同 | 所有顶点不同（起终可能相同） |
| -------- | -------- | ------------ | ---------------------------- |
| 简单通路 | ✅        |              |                              |
| 简单回路 | ✅        | ✅            |                              |
| 初级通路 | ✅        |              | ✅                            |
| 初级回路 | ✅        | ✅            | ✅                            |

!!! note

    * 四种特殊通路 都要求所有的边都不同
    * 回路需要起点终点相同
    * 初级通路要求处起终点以外的点都不同
